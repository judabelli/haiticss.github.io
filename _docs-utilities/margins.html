utilities > MARGINS

The margins utilities contain helper classes to force the CSS margin-top, margin-right, margin-bottom and margin-left properties.

Div with the class .u-mv--medium applied
{% highlight html %}
Div with the class .u-mv--medium applied
{% endhighlight %}

In the playground you can change throught the different medium spaces utilities, ineed you can find the same utilities for all spaces.

The mixin loop all defined spaces and outputs helper classes for each space and utility.

The mixin loops for all the nested colors map and output the different classes:

{% highlight scss %} {% include haiticss/src/utilities/_utilities__margins.scss %}{% endhighlight %}

So you can find utilities such as:

  • .u-mb-huge
  • .u-ml-small
  • .u-mt-tiny
  • .u-mv-none
  • .u-mh-medium
  • and all the others...
